# FoxEngine
FoxEngine is the first programming project that involved instance reading and YouTube tutorial assistance. Due to a lack of personal experience in graphics, this project provided crucial knowledge on how to enter the world of graphics programming. Using nothing more than the included libraries in Java, the imitation of a 3D world was achieved using raw mathematics running only on the CPU. There were some hiccups; even while watching The Cherno's YouTube series on how to make this specific game engine, updates to Java and features not working correctly in the video series forced some deviations to ensure the program worked correctly ... costing several hours of headaches and adjustments. However, this two-month endeavor was more than worthwhile.
